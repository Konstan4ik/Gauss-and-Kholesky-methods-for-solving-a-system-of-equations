# Сравнение методов Гаусса и Холецкого для решения СЛАУ

## Описание проекта

Проект реализует два метода решения систем линейных алгебраических уравнений (СЛАУ) с матрицей Гильберта:

- **Метод Гаусса** (схема единственного деления)
- **Метод Холецкого** (разложение Холецкого для симметричной положительно определённой матрицы)

Для каждой размерности матрицы от 1 до 13 вычисляется решение, замеряется время выполнения и оценивается норма невязки.

## Матрица Гильберта

Элементы матрицы Гильберта размера \(n \times n\) задаются формулой:
Матрица Гильберта является **симметричной** и **положительно определённой**, что позволяет применять к ней метод Холецкого. Однако она **плохо обусловлена**, особенно при больших \(n\), что приводит к росту погрешности численного решения.

## Реализованные методы

### 1. Метод Гаусса
- Прямой ход с исключением переменных
- Обратный ход для нахождения решения
- Вычисление нормы невязки

### 2. Метод Холецкого
- Разложение \(G = C \cdot C^T\) (нижняя треугольная матрица)
- Решение системы \(C y = B\) (прямой ход)
- Решение системы \(C^T x = y\) (обратный ход)
- Проверка положительной определённости матрицы

## Результаты

### Время выполнения

| Размерность n | Метод Гаусса (с) | Метод Холецкого (с) |
|---------------|------------------|---------------------|
| 1             | 2.20e-06         | 4.90e-06            |
| 2             | 4.30e-06         | 6.20e-06            |
| ...           | ...              | ...                 |
| 12            | 8.12e-05         | 6.72e-05            |
| 13            | 1.02e-04         | метод не применим   |

**Наблюдения:**
- Метод Холецкого быстрее при \(n \leq 12\)
- При \(n = 13\) матрица Гильберта перестаёт быть положительно определённой в рамках машинной точности

### Норма невязки

| n | Метод Гаусса | Метод Холецкого |
|---|--------------|-----------------|
| 1 | 0.00         | 0.00            |
| 2 | 5.00e-01     | 0.00            |
| 3 | 1.27         | 1.42e-14        |
| 4 | 2.50         | 3.22e-13        |
| 5 | 4.37         | 8.13e-12        |
| 6 | 7.16         | 4.69e-10        |
| 7 | 11.29        | 1.10e-08        |
| 8 | 17.35        | 3.29e-07        |
| 9 | 26.21        | 1.03e-05        |
| 10| 39.15        | 2.94e-04        |
| 11| 58.03        | 1.30e-02        |
| 12| 85.54        | 3.62e-01        |
| 13| 123.40       | ---             |

**Наблюдения:**
- Невязка метода Гаусса быстро растёт с увеличением \(n\)
- Метод Холецкого даёт существенно меньшую невязку, но также теряет точность при больших \(n\)
- При \(n = 13\) разложение Холецкого невозможно из-за нарушения положительной определённости
